Quadratic Equations

AP Master Notes - Ultimate CBSE Class 10 Guide

1. Introduction

A quadratic equation is an equation of degree 2. The highest power of the variable is always 2. Quadratic equations appear in mathematics, engineering, physics, architecture, computer graphics, economics and many real-life situations.

ax² + bx + c = 0

where a ≠ 0 b and c are real numbers x is the variable

Examples

  • x² + 5x + 6 = 0
  • 2x² − 3x + 1 = 0
  • 5x² − 20 = 0
  • 7x² + x − 9 = 0

Non Examples

  • x + 5 = 0
  • 3x³ + 4x + 1 = 0
  • √x + 2 = 0

2. Standard Form of a Quadratic Equation

Every quadratic equation can be written in the standard form:

a x² + b x + c = 0

where:

  • a = coefficient of x²
  • b = coefficient of x
  • c = constant term
  • a ≠ 0

Examples

  • x² + 5x + 6 = 0
  • 2x² − 3x + 1 = 0
  • 4x² − 25 = 0
  • 7x² + x − 8 = 0

Convert to Standard Form

Example 1:

x² + 5x = 6

x² + 5x − 6 = 0

Example 2:

(x + 2)(x + 3) = 0

x² + 5x + 6 = 0

3. Discriminant and Nature of Roots

One of the most important concepts in quadratic equations is the Discriminant.

D = b² − 4ac

The discriminant helps us predict the nature of roots without solving the equation.

Case 1 : D > 0

The equation has two distinct real roots.

Example: x² − 5x + 6 = 0 D = 25 − 24 = 1 Since D > 0, there are two different real roots.

Case 2 : D = 0

The equation has equal roots.

Example: x² − 6x + 9 = 0 D = 36 − 36 = 0 Hence both roots are equal.

Case 3 : D < 0

The equation has no real roots.

Example: x² + x + 1 = 0 D = 1 − 4 = -3 Since D is negative, no real roots exist.

Quick Summary

  • D > 0 → Two distinct real roots
  • D = 0 → Equal roots
  • D < 0 → No real roots
