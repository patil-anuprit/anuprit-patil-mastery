# Anuprit Patil's Board Mastery

**Plan Smart. Study Deep. Master Boards.**

A complete, fully functional premium educational website for CBSE Class 10 students. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies — deploying instantly as a static site on Netlify.

## Features

- **Home Page** — Hero section, board exam countdown (CBSE 2027), readiness score, feature showcase, student roadmap, motivational quotes, stats, testimonials
- **Study Planner** — Full task manager with add/edit/delete, priorities, due dates, subjects, streak tracker, daily & weekly goals, weekly calendar view, localStorage persistence
- **Notes Hub** — Complete CBSE Class 10 notes for Mathematics, Science, English, Social Science, and Information Technology — formulas, definitions, quick notes, live search
- **Quiz Zone** — Interactive MCQ quizzes (10 questions each) for all 5 subjects with instant feedback, answer explanations, score history, keyboard shortcuts (A/B/C/D, Enter)
- **About Anuprit** — Personal intro, vision, timeline, achievement badge system (15 unlockable badges), progress dashboard, contact form with localStorage

## Tech Stack

- **HTML5** — Semantic, multi-page structure
- **CSS3** — Custom properties (variables), CSS Grid, Flexbox, animations, dark/light mode
- **JavaScript (Vanilla)** — All interactivity, localStorage persistence, no libraries
- **Google Fonts** — Inter + Playfair Display

## Running Locally

```bash
# Using npx serve
npx serve .

# Or simply open index.html in a browser
open index.html
```
